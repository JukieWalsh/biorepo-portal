Configuring a Nautilus Data Source
==================================

The Nautilus data source provides the ability to define record labels.

.. code-block:: json

    {
        "labels": [1,2,3,4,5,6,7,8,9,18,19,20,21,22,23,24]
    }

Where each number in the labels key corresponds to the ID of a record label defined in the electronic Honest broker
